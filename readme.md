# Task Management Backend

### start server

git clone

```
npm i
```

- setup .env variables name in config file

```
npm start
```
- Local Server
    - http://localhost:5000

## API Routes

### User Routes
- /api/v1/signup (POST)
- /api/v1/login (POST)
- /api/v1/user (GET)

### Task routes
- /api/v1/task (GET)
- /api/v1/task/create (POST)
- /api/v1/task/:id (DELETE)
- /api/v1/task/update/:id (PATCH)
